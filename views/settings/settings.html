Data
Backup All Data
Downloads a full snapshot of all dashboard entries, time tracking records, and preferences as a single JSON file.
Restore from Backup
Replaces all current data with a previously exported backup file. This cannot be undone.
Reset All Data
Clears all financial entries, time tracking records, and saved preferences. This cannot be undone.
Remote Backup
GitHub Gist Sync
Connect a GitHub personal access token (with gist scope) and a Gist ID to back up dashboard data to a remote JSON file. The token is stored only in this browser.
