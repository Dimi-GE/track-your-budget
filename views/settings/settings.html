Data
Backup All Data
Downloads a full snapshot of all dashboard entries and preferences as a single JSON file.
Restore from Backup
Replaces all current data with a previously exported backup file. This cannot be undone.
Reset All Data
Clears all financial entries and saved preferences. This cannot be undone.
Currencies
Currencies & Regional
Manage the currencies available on the Dashboard. The regional currency is the default applied to every transaction (a foreign currency can only be chosen for Savings → Other). Marked with a star.
Exchange Rates
Savings Conversion
The approximate savings totals on Home and the Dashboard are converted into your regional currency (—) using these rates. Refresh pulls fresh live rates; a manual override wins for a currency when set (leave blank to use the live rate).
Remote Backup
GitHub Access Token
A GitHub personal access token with gist scope. Stored only in this browser and never written into any backup. Connect the token to unlock the backup Gist below.
Backup Gist
A secret GitHub Gist that stores a full snapshot of all your data — entries, currencies, rates, and preferences. Once connected, every Apply backs up automatically and each app open reconciles newest-wins; Backup / Restore force a push / pull. Restore replaces all current data.
