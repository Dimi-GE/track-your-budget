Data
Backup All Data
Downloads a full snapshot of all dashboard entries, time tracking records, and preferences as a single JSON file.
Restore from Backup
Replaces all current data with a previously exported backup file. This cannot be undone.
Reset All Data
Clears all financial entries, time tracking records, and saved preferences. This cannot be undone.
Currencies
Currencies & Regional
Manage the currencies available on the Dashboard. The regional currency is the default applied to every transaction (a foreign currency can only be chosen for Savings → Other). Marked with a star.
Exchange Rates
Savings Conversion
The approximate savings totals on Home and the Dashboard are converted into your regional currency (—) using these rates. Refresh pulls fresh live rates; a manual override wins for a currency when set (leave blank to use the live rate).
Remote Backup
GitHub Gist Sync
Connect a GitHub personal access token (with gist scope) and a Gist ID to back up dashboard data to a remote JSON file. The token is stored only in this browser.
Full Snapshot Backup
Connect to a dedicated GitHub Gist to keep a complete snapshot of all data — dashboard entries, currencies, time tracking, and preferences. While connected it syncs newest-wins each time you open the app, and Backup / Restore force a push / pull. Uses the same token as Gist Sync; create a separate secret Gist for this. Restore replaces all current data.
